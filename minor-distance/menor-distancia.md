# Menor distância

## Problema

Dado um array de inteiros não ordenado, encontre a menor distância entre os números e retorne os pares de números que possui a menor distância ordenado.

## Exemplo

```js
input: [1, 6, 8, 3, 434, 645, 10, 14, 18, 643, 20];
output: [
  [1, 3],
  [6, 8],
  [18, 20],
  [643, 645],
];
```
